## GUIA DA AULA
A estrutura para se utilizar laço FOR é bem simples no robot framework, segue o modelo:

> FOR  [condicional]
>
>    [AÇÃO]
>
> END

Existem outras keywords que pode nos ajudar nos testes com o laço FOR, estas seriam:
- Exit For Loop
- Exit For Loop If
