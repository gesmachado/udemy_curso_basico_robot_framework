## GUIA DA AULA

As principais keyword que serão utilizadas quando estivermos realizando testes de API com Robot Framework serão as seguintes:

Enviar requisição
- POST On Session
- PUT On Session
- PATCH On Session
- DELETE On Session

Início e fim de sessão
- Create Session
- Delete All Sessions
